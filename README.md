# 🌳 Santree

**Tumbuh Bersama Al-Qur'an**

Santree adalah platform gamifikasi pembelajaran Al-Qur'an untuk anak usia 8-12 tahun. Mengubah perjalanan ngaji jadi petualangan seru dengan quest, level, dan reward — tanpa screen time berlebihan.

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)
[![Open Source](https://img.shields.io/badge/Open%20Source-100%25-brightgreen)](https://github.com/zanjabilabanjarpatroman/santree)

## 🎯 Masalah yang Dipecahkan

- **Anak bosan ngaji** — Ngaji terasa kewajiban yang membosankan, tidak ada motivasi konsisten
- **Orang tua tidak tahu progress** — Tidak ada visibility perkembangan anak selain laporan bulanan TPQ
- **Ustadz kesulitan tracking** — Catatan manual di buku sering hilang, sulit pantau puluhan murid

## ✨ Fitur Utama

### 🕌 Mode TPQ
- Dashboard ustadz untuk kelola kelas
- Verifikasi setoran hafalan digital
- Leaderboard kelas
- Laporan otomatis ke orang tua

### 🏠 Mode Rumah
- Orang tua buat quest custom untuk anak
- Verifikasi dengan voice memo atau foto
- Pantau streak harian
- Reward & badge otomatis

### 🌙 Mode Event Musiman
- Quest spesial Ramadan, Syawal, dan momen istimewa
- Challenge keluarga
- Hadiah seasonal eksklusif

## 🎮 Cara Kerja: Phygital (Fisik + Digital)

Santree dirancang untuk **meminimalkan screen time anak**:

1. **Kartu Quest Fisik** — Download & print kartu quest (atau terima paket fisik)
2. **Anak Kerjakan** — Anak ngaji, hafal, shalat tanpa pegang HP
3. **Orang Tua Verifikasi** — Orang tua dengar bacaan, tap verifikasi di HP
4. **Level Naik** — Anak dapat EXP, badge, naik level dari Batu → Diamond

**HP hanya di tangan orang tua/ustadz**, bukan di tangan anak.

## 🃏 Kategori Quest

1. **Shalat & Ibadah Harian** — Subuh berjamaah, dzikir pagi-petang
2. **Hafalan Al-Qur'an** — Surat pendek, ayat pilihan, muroja'ah
3. **Doa Sehari-hari** — Doa makan, tidur, keluar rumah
4. **Adab & Akhlak** — Bantu orang tua, berbagi, jujur
5. **Materi Diniyyah** — Siroh nabawi, fiqih dasar, bahasa Arab
6. **Science** — Amati fase bulan, mengapa langit biru, wudhu & kesehatan
7. **Life Skill** — Rapikan kamar, masak sederhana, kelola uang saku

## 🛠 Tech Stack

- **Frontend:** Next.js 14 (App Router)
- **Backend:** Supabase (Database, Auth, Storage)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **License:** MIT
